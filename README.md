🚀 Smart Fleet and Inventory Intelligence System
An end-to-end data science–driven platform that enhances logistics and supply chain operations through predictive analytics, anomaly detection, and intelligent decision-making.

This system goes beyond traditional tracking by integrating fleet management and inventory intelligence into a single unified solution. It analyzes data from multiple sources such as GPS, traffic, weather, driver behavior, and inventory systems to predict delivery delays, detect operational anomalies, estimate time-to-disruption, and forecast inventory demand.

The platform includes:

🔮 Delay Prediction using machine learning

⚠️ Anomaly Detection for operational risks

⏳ Time-to-Event Analysis (survival modeling)

📦 Inventory Demand Forecasting and Stock-Out Prediction

📊 Interactive Dashboard for real-time monitoring and insights

Built with a FastAPI backend and React + Tailwind frontend, the system enables organizations to move from reactive tracking to proactive, data-driven decision-making.
